 Infrastructure HIGH
75%
Module-level singleton en client.ts. globalThis.__prisma pattern para hot-reload dev.
R-03
JWT no revocable post-logout
Token 7d sin blacklist. Sesión robada no puede invalidarse server-side.
 Security HIGH
60%
Token versioning en DB (jti + version col). O reducir expiración a 1h + refresh token.
R-04
Email no verificado puede acceder
emailVerifiedAt en DB pero no chequeado. Cuentas fantasma o no confirmadas.
 Auth MED
70%
Agregar check en AuthenticateUser: if !emailVerifiedAt → return error email_not_verified.
R-05
Event bus ausente = side effects perdidos
Operaciones de negocio no emiten DomainEvents. Alertas, recomendaciones y IA no se disparan.
 Architecture MED
100%
Implementar Trigger.dev o simple in-process EventEmitter como primer paso. Luego migrar a queue.

ConnectorCredentials encriptación no verificada

 Security MED

 Domain Logic MED

 Domain Logic LOW
