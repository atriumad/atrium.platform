The signature marker-highlight swipe behind an emphasised word — Atrium's core emphasis device (use instead of bold/italic for keywords).

```jsx
<h2>Simple software for <Highlight>better Business</Highlight></h2>
<p>Marketing that actually <Highlight color="mint">converts</Highlight>.</p>
```

`color`: `amber` (default), `mint`, or `teal` (which flips text to mint). Wraps across line breaks. Never used with emoji.
