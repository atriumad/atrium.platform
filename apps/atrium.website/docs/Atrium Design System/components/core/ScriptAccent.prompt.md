Handwritten accent word (script font) dropped into a headline for warmth — pair with bold sans/serif words.

```jsx
<h1>We're <ScriptAccent>humans</ScriptAccent></h1>
<h2>Welcome to <ScriptAccent color="var(--mint-400)">atrium</ScriptAccent></h2>
```

Underlined by default (`underline={false}` to turn off). `color` defaults to inherit.
