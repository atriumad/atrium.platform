Flat, color-led surface card — Atrium uses the field color for hierarchy, not shadows.

```jsx
<Card tone="mint" hover>
  <h3>Graphic Design</h3>
  <p>Brand systems, packaging, the works.</p>
</Card>
<Card tone="light" bordered>…</Card>
```

`tone`: `light | cloud | mint | amber | teal`. Set `hover` for a lift on interactive cards, `bordered` for a hairline on light cards.
