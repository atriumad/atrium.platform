Atrium logo — the α monogram, lowercase wordmark, or horizontal lockup. Recolorable for any field.

```jsx
<Logo variant="wordmark" color="var(--teal-800)" height={28} />
<Logo variant="mark" color="var(--mint-400)" height={40} />   {/* on deep teal */}
<Logo variant="lockup" color="var(--mint-400)" height={32} />
```

`color` accepts any token/hex (applied via CSS mask). Set `assetBase` to the relative path of `assets/logos` from your page (default `../../assets/logos`).
